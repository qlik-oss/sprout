# @qlik/sprout-cli

The goal of this package is to provide a CLI with different kind of utility that are useful for users of the design system sprout.

## tokens commands

> sprout-cli token --format=css --source tokens --output dist

This command will look for a tokens folder and will build a CSS file from it.

The command support the following options:

- v/verbose
- s/source <string>
- f/format <string>
- o/output <string>
