# Migration from MUI to Sprout CSS React

This guide outlines how to migrate components and styling from **MUI** to **Sprout CSS React**.

## sx props

Restyling a component is a violation, you must consider each Sprout component as a Lego brick that you can't change anything, even adding margin. This is the only way to guarantee consistency and ownership of those components. Overrides makes update impossible.

The different technics to work with sprout components:

### Solution 1: Wrap the component

Instead of adding margin to a "button" you can do wrap it and add the margin on this wrapper:

```typescript
<span className={classNames('m_xs')}><Button onClick={} label="label"/></span>
```

### Warning: MarginLeft, PaddingRight, and other physical properties brakes RTL support

First you should rely on `classNames` all the time even for small layouting.

This example is wrong as it breaks RTL. Also the spacing are incorrect, you should always rely on the spacing and density [tokens](https://internal.qlik.dev/design/sprout/design-tokens/list-of-all-tokens/#spacing)

```typescript
<Box
  sx={{
    paddingLeft: 1,
    paddingRight: 1,
    marginBottom: "2px",
  }}
>...</Box>
```

This kind of layout should be done by using our
[padding](/?path=/story/css-sprout-css-modules--typography) and margin utility classes
which are named using left/right top/bottom but internally are using logical properties:

```typescript
<div className={classNames("flex", "border_box", "flex_row", "pl_xs", "pr_xs", "mb_xs")}>...</div>
```

An other important point if the design has 1px, it's incorrect and you should request for an update. You should never need 1px in spacing.

## Typography

You can rely on our classNames for [typography](/?path=/story/css-sprout-css-modules--typography):

```typescript
import { classNames } from '@qlik/sprout-css'

<h1 className={classNames("font_heading_m", "text_default")}>my title</h1>
```

## Box

Box doesn't exist in sprout-css. You should rely on our [flex](/?path=/story/css-sprout-css-modules--flexbox) and [grid](/?path=/story/css-sprout-css-modules--grid) utilities instead.

```typescript
import { classNames } from '@qlik/sprout-css'

<div className={classNames("flex", "flex-col", "border-box")}></div>
```

## Grid

sprout-css provides a CSS Grid system supporting up to 16 columns and 4 rows, with responsive grids coming soon.

```typescript
import { classNames } from '@qlik/sprout-css'
<div className={classNames("grid", "grid-cols-12", "gap-m")}>
  <div className={classNames("col-span-2")}></div>
  ...
</div>
```
