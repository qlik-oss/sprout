# Table - Row height

Customize the height of your rows.

## Static row Height: rowHeight

The `rowHeight` prop allows you to set a static height for all rows in the table. This is useful when you want to maintain a consistent row height across your table.

```tsx
import { Table } from "@qlik/sprout-table";

return <Table rowHeight={45} rows={rows} columns={columns} />;
```
