# useStyleOverride

The `useStyleOverride` hook allows you to generate new token values and apply them to a part of your page.

## API

`useStyleOverride` accept an object with the provided tokens to override as key-value pairs as a first argument.
The second argument is an optional string to target a specific element. It's passed to the `document.querySelector` method.

```ts
import { useStyleOverride } from "@qlik/sprout-react-hooks";
```
