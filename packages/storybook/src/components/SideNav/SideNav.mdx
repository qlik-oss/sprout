## Usage

```tsx
import { List, ListItem, SideNav } from "@qlik/sprout-react";

<SideNav.Nav>
  <SideNav.Section label="Files">
    <List gap="density-s">
      <ListItem hasPadding={false} interactive>
        <SideNav.Link
          href="#upload"
          icon={<CloudUploadIcon />}
          label="Uploads"
        />
      </ListItem>
      <ListItem hasPadding={false} interactive selected>
        <SideNav.Link
          href="#recent"
          aria-current="page"
          icon={<HistoryIcon />}
          label="Recent"
        />
      </ListItem>
    </List>
  </SideNav.Section>
</SideNav.Nav>;
```

> **Note:** `ListItem` requires `hasPadding={false}` when used inside `SideNav` for correct composition. This prop does not appear in the source code panel.

## Compact variant

The `variant="compact"` prop can be applied to `SideNav.Nav`, `SideNav.Item`, and `SideNav.Link` to switch to a condensed layout. In compact mode, icons can also use `iconSize="large"` for a larger touch target.

```tsx
<SideNav.Nav variant="compact">
  <SideNav.Section label="Files">
    <List gap="density-s">
      <ListItem hasPadding={false} interactive>
        <SideNav.Link
          variant="compact"
          href="#upload"
          icon={<CloudUploadIcon />}
          label="Uploads"
        />
      </ListItem>
      <ListItem hasPadding={false} interactive>
        {/* icon-only link */}
        <SideNav.Link
          variant="compact"
          href="#recent"
          icon={<HistoryIcon />}
          aria-label="Recent"
        />
      </ListItem>
    </List>
  </SideNav.Section>
</SideNav.Nav>
```

## Accessibility

- [W3/WAI/ARIA/APG navigation landmark](https://www.w3.org/WAI/ARIA/apg/patterns/landmarks/examples/navigation.html)

HTML structure

- `SideNav.Nav` renders a `nav` element.
- `SideNav.Section` renders a `section` element with `aria-labelledby` pointing to its heading when a `label` is provided. The heading level defaults to `h2` and can be changed via the `level` prop.
- `SideNav.Link` renders an `a` element and wraps `SideNav.Item` internally.
- `SideNav.Item` renders a `span` element.

Keyboard navigation

- Use `Tab` / `Shift+Tab` to move focus between links.
- The currently active link should carry `aria-current="page"`.
- When a link opens an external destination, `isExternal` adds a visual indicator; pair it with a descriptive `aria-label` when no visible label is present.

Please see the [List](?path=/docs/components-list--docs) documentation for further details on `List` and `ListItem`.
