# AlertBanner

Alerts are short messages that shows feedback, information, warnings, or critical errors.

## Usage

```tsx
import {
  AlertBanner,
  type AlertBannerProps,
} from "@qlik/sprout-react";

<AlertBanner
  variant="info"
  content="Alert content"
  onClickDismiss={() => {}}
  action={<></>}
/>;
```

## Accessibility

This component use a `div` html tag with the `role=status` and `aria-live="polite"` attribute to announce the alert to screen readers.
You can switch to a real alert by overriding those two properties like this:

```tsx
<AlertBanner
  role="alert"
  aria-live="assertive"
  variant="error"
  content="Urgent alert"
/>
```
