# ThemeContext / useTheme

You can read if the current device is touch-enabled by using the `isTouch` property from the context value:

```tsx
import { useTheme } from "@qlik/sprout-react";

const MyComponent = () => {
  const { isTouch } = useTheme();

  return <div>{isTouch ? "Touch device" : "Non-touch device"}</div>;
};
```
