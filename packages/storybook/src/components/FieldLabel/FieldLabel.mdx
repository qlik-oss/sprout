# FieldLabel

FieldLabel is the primary label component for form fields. It renders a semantic `label` element, supports an optional indicator, and can display an information icon next to the text. FieldLabel directly inherits properties from the [HTML label element](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/label#attributes).

```tsx
import { FieldLabel, Popover } from "@qlik/sprout-react";

<FieldLabel label="Label" optional />

<FieldLabel label="Label" infoIconPopover={<Popover />} />
```
